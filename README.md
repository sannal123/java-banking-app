# Java Banking Application

A console-based Banking Management System developed using **Java, JDBC, and MySQL**.

## Features

- Create Account
- Login
- View Balance
- Transfer Money
- MySQL Database Integration
- JDBC PreparedStatement
- Transaction Management (Commit/Rollback)

## Technologies Used

- Java
- JDBC
- MySQL 8
- Eclipse IDE
- Git
- GitHub

## Project Structure

```
src/
 └── banking/
      ├── DBConnection.java
      ├── bank.java
      └── bankManagement.java
```
